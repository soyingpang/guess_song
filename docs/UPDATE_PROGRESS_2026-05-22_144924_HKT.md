# 更新進度 - 2026-05-22 14:49:24 HKT

## 今次更新重點

- 修正手機玩家首次連線失敗時，會跳回「我在現場 / 我不在現場」選擇畫面的問題。
- 現在玩家輸入名字並選擇模式後，如暫時連不到主持房間，會保留已選模式並留在連線狀態。
- 手機端會自動重試，不需要會友重新選擇在場狀態。
- 連線錯誤文字更清楚：
  - 找不到主持房間時提示確認控制台保持開住。
  - 手機網絡問題時提示同步服務暫時連不到。

## 原因

- 舊流程把「第一次連線失敗」當成加入失敗處理，會解鎖模式並重新顯示選擇按鈕。
- 這會令手機看起來像是自動跳回前一步，實際上是 PeerJS 未能連到主持房間或網絡暫時未通。

## 下一步建議

- 若手機仍一直自動重連，請檢查主持控制台是否保持開住，以及控制台右側房間狀態是否顯示固定房間已建立。
