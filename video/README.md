# Video files

Put only video files your church is allowed to use here, for example self-recorded worship clips, licensed MP4/M4V/MOV/WEBM/OGV files, or files released with clear permission.

In the host song form, use paths like:

```text
./video/song-name.mp4
```

The app keeps the field name `audioUrl` for old library compatibility, but MP4 / video paths are supported. When a song has a local / authorized media URL, the app plays that file instead of YouTube, so the front display can show visuals without YouTube ads.
