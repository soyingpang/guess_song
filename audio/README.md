# Audio files

Put only audio files your church is allowed to use here, for example self-recorded clips, licensed MP3/M4A/WAV/OGG files, or files released with clear permission.

In the host song form, use paths like:

```text
./audio/song-name.mp3
```

When a song has a local / authorized audio URL, the app plays that file instead of YouTube, so the front display can run without YouTube ads.
